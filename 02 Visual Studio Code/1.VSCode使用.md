# 1.2 运行和调试
- 配置 `launch.json` 文件进行调试
	- `Python` 也可以右上角一键调试
	- `C/C++` 可以下方 `CMake Tools` 一键调试
- 学会合理地打断点
- 通过监视变量来判断哪里有问题
	- 变量的监视前提是断点打好
	- 监视变量可以一个一个变量来看
	- 改变变量值的时候记得使用 `控制变量法`

# 1.3 使用技巧
- `ctrl` + `P`：搜索文件
- `ctrl` + `shift` + `P`：命令面板
- `Alt` + `上/下箭头`：快速交换两行
- `ctrl` + `shift` + `K`：快速删除一行
- `ctrl` + `C`：快速复制光标所在行
- `ctrl` + `X`：快速剪切光标所在行
- `ctrl` + `V`：在光标所在行的上一行插入粘贴内容
- `ctrl` + `\`：向右拆分编辑器
- `ctrl` + `shift` + `I`：快速排版代码
- `ctrl` + `shift` + `C`：快速在当前目录中打开终端
- `F5`/`ctrl` + `F5`：快速运行文件或者调试文件
- `ctrl` + \` ：新建集成终端
